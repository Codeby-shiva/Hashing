# Hashing

# What is hashing ?
Hashing is used for searching item in constant time O(1) .

We used a hash function for store data in return index value by hash function, and hash value/index number calculate according data .




# How it is working ?
Create a hash function according own choice, and put a data in hash function and this function return a hash value where data will be store,
and when we want to search the data so, simply we put the data in that hash function and that return a hash value and we will search there,
if data found so return either return not found.



# Use and Benifitis
This hashing used for searching data in constant O(1) time complexity.

By the use of hash function we can easily find the data from a big array.
